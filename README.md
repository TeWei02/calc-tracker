# Calc Tracker 微積分練習與錯題管理系統

一個幫助大學轉學考生管理微積分練習題與錯題的 Web 應用程式。  
使用 Flask + SQLite 實作，記錄每題作答情況並提供章節統計與錯題清單。

## 功能

- 新增題目：輸入題目標題、章節、來源、難度與備註。
- 題目列表：依時間或章節列出所有題目，練習時可以標記「答對／答錯」。
- 錯題統計：自動整理錯兩次以上的題目，方便集中複習。
- 章節統計：顯示各章節作答次數與正確率，幫助掌握弱項。

## 技術棧

- Backend：Python, Flask
- Database：SQLite（透過 Flask-SQLAlchemy 操作）
- Frontend：HTML, Bootstrap, Jinja2

## 開發動機

為了準備大學轉學考（特別是微積分），希望有一個系統可以：
- 系統性整理各章節題目與來源（課本、考古題、參考書等）。
- 記錄每次練習結果，追蹤自己的弱點與進步。
- 未來可擴充至其他科目（計概、離散等）或加入 AI 輔助提示功能。
